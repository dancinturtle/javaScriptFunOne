Output of the program can be seen using "Inspect Element" and clicking on the "Console tab"

Slots game where 1 game = 1 qtr.
Have 1 in 100 chance of winning.
Winnings can range from 50--> 100 quarters.
Use Math.random + Math.floor to pick random # of coins if they win.
Return new quarter balance if they win, Return 0 if they used all the quarters.
